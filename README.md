## ⚙️ Installation

### 1️⃣ Clone Repository

```bash id="cmd1"
git clone https://github.com/dudekularehaman2006/CyberShield-AI.git
cd CyberShield-AI
```

---

### 2️⃣ Setup Backend

```bash id="cmd2"
cd backend
pip install -r requirements.txt
```

---

### 3️⃣ Add API Key

```env id="env1"
GROQ_API_KEY=your_api_key_here
```

---

### 4️⃣ Run Backend

```bash id="cmd3"
python app.py
```

---

### 5️⃣ Run Frontend

```bash id="cmd4"
open frontend/index.html
```
